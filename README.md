# AuSync

Secure cloud storage powered by Telegram.

AuSync turns your Telegram account into an unlimited, secure cloud storage drive. Built with Tauri, Rust, and React.

## Features
- Unlimited cloud storage via Telegram
- High-performance virtual scrolling
- Media streaming (video/audio)
- PDF viewer
- Drag & drop uploads
- Cross-platform (Windows, macOS, Linux)
